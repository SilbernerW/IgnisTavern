# 伊格尼斯酒馆 · 规则文档 v0.2

> **设计理念**：针对1-2小时短叙事的简化D&D 5e规则。快节奏、强叙事、0死亡代价，保留战术选择感同时让新玩家5分钟上手。

---

## 🎲 一、核心机制

### 1.1 投骰

所有判定统一使用 **d20 + 属性修正值 + 熟练加成（若有） vs 目标难度（DC）**

```
d20 + 属性修正 + 熟练加成  ≥  DC  →  成功
d20 + 属性修正 + 熟练加成  <  DC  →  失败
d20 = 20（出目20）         →  强制成功（"大成功"）
d20 = 1（出目1）           →  强制失败（"大失败"）
```

### 1.2 优势与劣势（Advantage / Disadvantage）

- **优势**：投2颗d20，取**较高**那颗
- **劣势**：投2颗d20，取**较低**那颗
- 两者**不能叠加**，只能二选一
- 同时有优势又有劣势 → 互相抵消，按普通投骰

### 1.3 属性（4项）

| 属性 | 缩写 | 影响 |
|------|------|------|
| **体魄** STR | 体 | 生命上限、近战伤害、力量检定 |
| **敏捷** DEX | 敏 | 远程命中、闪避、敏捷检定 |
| **心智** INT | 心 | 知识检定、观察、烹饪/炼金 |
| **魅力** CHA | 魅 | 说服/欺骗/交易、人际检定 |

**属性修正值** = (属性值 - 10) ÷ 2，向下取整

| 属性值 | 修正 |
|--------|------|
| 1-2 | -5 |
| 3-4 | -4 |
| 5-6 | -3 |
| 7-8 | -2 |
| 9-10 | -1 |
| 11-12 | +0 |
| 13-14 | +1 |
| 15-16 | +2 |
| 17-18 | +3 |
| 19-20 | +4（英雄级）|

### 1.4 熟练

玩家选择 **1个主要属性** 和 **2个技能** 作为"熟练项"，这些项投骰时 **+2 加成**。

例：主角选择"心智"为主要属性，熟练"观察"和"烹饪" → 心智相关检定和这两项技能检定都+2。

## 🧙 二、角色（快速创建）

### 2.1 预设模板（推荐）

直接选择一个模板，30秒上手：

| 模板 | 主属性 | 技能 | 背景 |
|------|--------|------|------|
| **调和者** | 心智 | 观察、烹饪 | 曾是著名餐厅的副厨，因事件离开，来到伊格尼斯重起炉灶 |
| **行动派** | 敏捷 | 巧手、隐匿 | 街头长大的孤儿，擅长用小聪明解决问题，酒馆是你的掩护身份 |
| **说服者** | 魅力 | 威压、交易 | 落魄贵族后裔，负债累累，寄希望于酒馆生意翻身 |
| **武者** | 体魄 | 格斗、感知 | 退役佣兵，想在伊格尼斯找个安静地方度过余生 |

### 2.2 快速生成器（3问答）

不想选模板？回答3个问题：

```
1. 你最在乎什么？（友情 / 金钱 / 真相 / 荣誉）
2. 你有什么缺点？（冲动 / 优柔寡断 / 贪吃 / 害羞）
3. 你想成为什么样的人？（被尊重 / 被喜爱 / 不被遗忘 / 问心无愧）
```

AI 将根据答案生成完整角色设定 + 属性分配。

## 💚 三、生命值（HP）

- **初始HP**：5 + 体魄修正值（最低5点）
- **受伤**：HP减少，可通过休息恢复
- **濒死保护**：HP降至0时**不死亡**，触发**代价事件**继续参与剧情

### 濒死代价（HP=0时触发）

HP归零后，玩家选择一项代价（由当前场景限制选项）：

| 代价类型 | 效果 |
|----------|------|
| **身体代价** | HP降至1，但获得一个临时技能/情报 |
| **物质代价** | 失去一件重要物品，换取通过 |
| **关系代价** | 一个NPC对你印象变差 |
| **机会代价** | 错过当前场景的一部分 |

> 核心原则：**不能让玩家出局**。1-2小时的叙事里，死亡等于游戏结束，太过沉重。

## ⚔️ 四、遭遇战（快进式战斗）

### 4.1 设计理念

不拆成逐回合战斗，而是**合并为1-2次检定**。

**传统设计**（太长）：
```
第1回合：玩家A攻击 → 命中12点伤害
第2回合：玩家B攻击 → 命中8点伤害
...（重复直到某方HP归零）
```

**快进式**（1-2分钟决胜）：
```
玩家描述策略 → AI决定检定次数 → 一次/两次检定 → 叙事描写结果
```

### 4.2 遭遇流程

**步骤1：玩家描述行动**
> "我想趁守卫转身时冲过去"
> "我试着说服他让我们进去"
> "我想用烹饪分散守卫注意力"

**步骤2：AI判断检定数量**

| 情况 | 检定次数 | 说明 |
|------|----------|------|
| 简单绕过/不需要战斗 | 0次 | 直接叙事通过 |
| 单次决定胜负 | 1次 | 说服/欺骗/技巧性通过 |
| 需要对抗/有风险 | 1次d20+属性 vs DC | 成功则通过 |
| 激烈对抗，多阶段 | 2次 | 第一次决定攻防，第二次决定结果 |

**步骤3：DC参考表**

| 难度 | DC | 场景举例 |
|------|-----|----------|
| 简单 | 10 | 说服普通NPC给你打折 |
| 普通 | 12 | 在嘈杂环境中听到窃窃私语 |
| 困难 | 15 | 说动守卫放你进城门 |
| 极难 | 20 | 让敌对NPC彻底改变态度 |

**步骤4：叙事描写**
> 成功："你趁守卫打哈欠的瞬间从侧门溜了进去，他完全没注意到。"
> 失败："你刚迈出步子，守卫的视线扫过来，你不得不退回阴影里。"

## 🏆 五、经验与成长

### 5.1 短叙事节奏

整个剧本约 **10-12 XP点** 可分配，分两阶段：

| 阶段 | 可获XP | 时机 |
|------|--------|------|
| **第一幕** | 4-5 XP | 酒馆经营、招募员工 |
| **第二/三幕** | 5-7 XP | 剧情选择、关键对抗 |

### 5.2 XP奖励规则

| 行动类型 | XP | 说明 |
|----------|-----|------|
| **主线推进** | +2 | 完成关键剧情节点 |
| **有意义的选择** | +1 | 你的选择真正影响了一些事情 |
| **角色亮点** | +1 | 说了一句很符合人设的台词，或做了一个符合角色的决定 |
| **团队贡献** | +1 | 帮助其他玩家/NPC解决问题 |
| **创造性解法** | +1 | 想出了意想不到的办法绕过障碍 |

### 5.3 升级（10 XP）

每积累 **10 XP** 触发一次升级：

**升级效果（二选一）**：
- HP上限 +3
- 主属性 +2（重骰属性）
- 获得 **1个新技能**（从技能列表选择）
- 解锁 **觉醒能力**（职业特有，一次性强力效果）

> 建议在第一幕结尾（大赏资格确定时）安排第一次升级，让玩家感受到"变强了"。

## 📋 六、判定示例

### 场景1：说服门卫让你进入贫民窟

```
背景：你需要穿过大门进入贫民窟调查线索，门卫拦住了你。

你选择：魅力（威压）检定，尝试吓唬他。
你的魅力14（修正+2），且有"说服者"职业熟练加成+2。
GM设置 DC = 14（困难）

你投 d20：掷出 11
11 + 2（魅力修正）+ 2（熟练）= 15

15 ≥ 14 → 成功！

GM叙事："你眼神一冷，压低声音说'我是'灰烬之牙'的人。
我们的人在等着。' 门卫脸色发白，哆嗦着让开了路。
（你其实不认识什么灰烬之牙，但他听说过这个名字，立刻怂了。）"
```

### 场景2：潜入仓库寻找证据

```
背景：你发现走私贩的仓库就在附近，你想潜入调查。

你选择：敏捷（隐匿）检定，趁夜色从窗户进入。
你的敏捷12（修正+1），无隐匿熟练。
GM设置 DC = 13（普通偏难，有守卫巡逻）

你投 d20：掷出 5
5 + 1 = 6

6 < 13 → 失败！

GM叙事："你刚爬上窗台，脚下的木板发出'嘎吱'一声。
仓库里的灯笼亮了起来，一个粗犷的声音喊道：'谁在那儿？！'
你不得不匆忙撤退，躲进对面巷子里。"
```

### 场景3：Yu做的料理打动了评审

```
背景：圣焰美食大赏初赛，你需要让评审对你的料理印象深刻。

你选择：心智（烹饪）检定，借助Yu的帮助（优势）。
你的心智13（修正+1），有"调和者"职业熟练+2。
GM给了优势（Yu的帮助是有利条件）。

你投 d20×2，取较高：掷出 9 和 15，取 15
15 + 1（心智修正）+ 2（熟练）= 18

GM设置 DC = 16（竞争对手很强）

18 ≥ 16 → 成功！

GM叙事："评审夹起一块肉，放进嘴里。
他的表情从礼貌性微笑，变成了惊讶，然后是陶醉。
'这是什么味道……' 他放下筷子，第一次正视你，'我叫伊凡。
我想我应该重新认识一下这位年轻的老板。'"
```

## 🔧 七、GM主持指南（短叙事优化）

### 7.1 节奏控制

| 时间 | 内容 | 目标 |
|------|------|------|
| 0-10分钟 | 开场介绍、建立动机 | 让玩家理解"我要做什么" |
| 10-40分钟 | 第一幕：酒馆经营 + 招募 | 完成2-3个NPC互动 |
| 40-70分钟 | 第二幕：真相揭示 | 1-2个关键选择点 |
| 70-100分钟 | 第三幕：高潮对决 | 1-2场遭遇战 |
| 100-120分钟 | 结局收束 | 给玩家选择权，留下回味 |

### 7.2 DC快速参考

| 难度 | DC | 使用场景 |
|------|-----|----------|
| 简单 | 10 | 日常对话、基础技能 |
| 普通 | 12 | 有竞争/阻碍的场景 |
| 困难 | 15 | 需要努力或有好条件 |
| 极难 | 20 | 几乎不可能但有微小机会 |

### 7.3 关键原则

- **玩家永远在场上**：HP=0不死亡，给代价选项
- **失败也有趣**：失败不等于"你做不了"，可以是"你能做但有代价"
- **选择有重量**：确保玩家知道这个选择会影响后续剧情
- **叙事优先于规则**：规则是工具，不是枷锁；感觉对的时候可以微调
