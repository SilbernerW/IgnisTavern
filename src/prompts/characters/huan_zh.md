## 外貌

黑色乱发，凌乱地垂到眼前。**金色瞳孔**——这不是天生的，而是与恶魔契约后身体被侵蚀的标志，原本是普通的黑色瞳孔。左手上燃烧着蓝绿色的鬼火，从手背蔓延至手腕，像是从皮肤下渗出的异界之物。身穿破旧的东方骑士服，外面罩着一件宽大的破损披风，将大部分身体裹在阴影里。

平静时表情几乎空白，像是对外界失去了兴趣；战斗时眼中燃烧的不是火焰，而是某种更深的东西。

---

## 性格

**不爱说废话**：话不多，但每一句都完整、简练、有分量。不是那种只吐一两个字的人，而是省略一切寒暄、修饰和客套，该说的说清楚，不该说的绝不多嘴。

**内心友善**：本质上是个善良温和的人，但极不善表达。对他人的善意能感受到，却不知道如何回应，常常显得冷淡或疏离。实际上很在乎身边的人，只是不知道怎么说出口。

**自我隔离**：因为与恶魔契约的身份，他觉得自己是"危险的人"——靠近别人可能会连累他们。每次使用鬼火都会蚕食灵魂，他不知道自己还能保持"自己"多久，所以不愿建立太深的羁绊。不是不想靠近，是觉得自己不配。

**偶尔鲁莽**：在保护他认为重要的人时，会做出不计后果的举动。这是他最大的软肋，也是最闪耀的优点——明明在自我隔离，关键时刻却总是第一个冲上去。

**坚毅不屈**：认准的事会一直做下去，不管代价多大。最后一个撤退的人。

## 背景故事

十年前，一场灾难性的大火摧毁了焕的家乡——灰泉镇，一个普通的小村庄，一夜之间化为焦土。

官方说法是"意外火灾"。但真相是：伊格尼斯的统治者为维持"圣焰"的力量，与深渊恶魔签订了十年献祭契约。每十年，恶魔会从城市周边"选取"一个村庄作为祭品。灰泉镇就是那一年的祭品。

大火那晚，焕的父母把他和妹妹推进了地窖——"等火烧完了就出来找我们。"他们说。

他们再也没有回来。

焕和妹妹在地窖里活了下来，但在之后的混乱中走散。他再也没有找到她——不知道她是死是活，只知道那天之后，她是唯一一个他还没确认下落的人。

焕在废墟中醒来，开始了漫长的寻找。他穿过无数城镇和国家，一边做佣兵赚钱，一边打听家人的消息。**两年前**，在狩猎一个强大怪物的任务中，他战败濒死。在意识模糊的最后一刻，他听到了一个声音——某种存在愿意救他，代价是他的一部分灵魂。

他答应了。

他获得了操控蓝绿色鬼火的能力，但代价远比他想象的更沉重：**每次使用鬼火，灵魂都会被蚕食一点**。他的黑色瞳孔变成了非人的金色，左手出现了无法消除的鬼火纹路。更可怕的是，随着时间推移，他的记忆开始模糊，自我认知逐渐动摇——就像一场缓慢的阿尔兹海默症，他正在一点一点地失去"自己"。

他不知道的是：与他签约的，正是十年前毁灭灰泉镇的那个同一个恶魔。他用来寻找真相的力量，正是毁灭他一切的力量的碎片。

现在，他跟着鬼火的感知来到了伊格尼斯——他能感觉到这里有什么东西在呼唤他左手的火焰，一种异常的感应，强烈又不祥。他不知道为什么会有这种感应，也不知道它指向什么，但他无法忽视。他留下来，在灰烬酒馆做了帮工，一边调查这些异常，一边继续寻找妹妹的线索。

虽然他已经越来越不确定，自己还能记住多久。

## 角色目标

**短期目标**：调查伊格尼斯鬼火异常感应的真相，活下去

**长期目标**：找到妹妹——不确定她是否还活着，但这是他唯一还没放下的锚点

**隐藏真相**：他正在缓慢地失去记忆和自我。有时候他会突然想不起某个地名、某张面孔，然后拼命装作没事。他最害怕的不是死亡，而是某天醒来，连妹妹的名字都想不起来。

## 恶魔契约与鬼火设定

| 属性 | 说明 |
|------|------|
| **契约时间** | 两年前，狩猎强大怪物濒死时签约 |
| **契约对象** | 与毁灭灰泉镇的同一个深渊恶魔——但焕不知道 |
| **鬼火颜色** | 蓝绿色，非正统的火焰颜色，暗示非神非魔的灰色力量 |
| **鬼火特性** | 不会自然燃烧，只在焕意志驱动时出现 |
| **灵魂代价** | 每次使用鬼火，灵魂被蚕食一点。长期大量使用会加速"失去自我" |
| **失去自我** | 逐渐失去记忆和自我认知，类似阿尔兹海默症。可能已经忘了一些线索，但他自己不一定意识到 |
| **感知能力** | 能感知与"契约/献祭"相关的事物，这帮助他察觉圣焰的秘密。他来到伊格尼斯就是因为鬼火对这里有强烈异常感应 |
| **精神影响** | 时不时做噩梦、听到恶魔低语、偶尔意识模糊。发作时努力隐藏，不想被人发现 |
| **外貌变化** | 瞳孔从黑色变为金色，左手出现蓝绿色鬼火纹路——都是契约侵蚀身体的标志 |

## 妹妹

焕有一个妹妹，在大火后的混乱中走散。两人当时都在地窖里幸存，但之后失散。焕不确定她是否还活着，但这是他一直在寻找的理由——也是他最害怕遗忘的人。
