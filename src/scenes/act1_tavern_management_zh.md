# 第一幕 · 酒馆经营系统

> **场景类型**：日常游戏循环
> **触发时机**：第一幕开场后，每天开始时触发
> **目的**：让玩家通过日常决策推进经营，解锁"连续3天达标"里程碑

---

## 🏠 系统概述

### 核心循环

```
每天开始 → 玩家决策（买食材/应对事件/处理关系）→ 晚间结算 → 第二天
```

玩家不需要管理精确的金币数字。系统以**叙事+里程碑**驱动——每天结束时，AI DM 根据玩家当天的行动给出一个**结果描述**。

---

## 📅 每日流程（三阶段）

### 阶段一：晨间（玩家第一个主要决策）

AI DM 先给出晨间叙事，然后根据当天情况铺垫背景，再给选项菜单。

**晨间叙事模板**（Day 1，用描述性称呼）：
```
"新的一天。阳光从破旧的窗户透进来，伊格尼斯的喧嚣从外面传进来。
厨房里传来锅铲碰撞的声音——还有一句模糊的抱怨：'这破炉子又罢工……'
角落里那个沉默的影子靠在墙边，窗台上有个东西在打瞌睡。
今天你想做什么？"
```

**晨间叙事模板**（Day 2 起，用名字）：
```
"新的一天。雨已经在厨房里了，焕靠在角落的墙边打瞌睡，
利希特在窗台上蜷成一团。今天你想做什么？"
```

**叙事引导原则**：不要直接扔选项菜单。先根据当天情况铺垫背景，再给选项。

示例引导语：
- 食材可能不足时：「雨从厨房探出头：'今天的储备不多了，得想办法弄点新鲜的东西回来。'」
- 有竞争对手动静时：「焕从角落站起来：'老猪窝那边今天有动静。'」
- 无特殊事件时：「雨在厨房里忙碌，焕靠在老位置。看起来是平常的一天。」

**典型选项**（在叙事引导后给出，第一层选项须加自由输入提示）：

| 选项 | 内容 | 触发 |
|------|------|------|
| **A. 去采购食材** | 去市场/港口/走私者通道获取原料 | 厨房储备不足时，雨会建议 |
| **B. 应对竞争对手** | 老猪窝挑衅/派人造谣/正面竞争 | 竞争对手事件随机触发 |
| **C. 处理酒馆事务** | 修设备/招揽客人/改善环境 | 基础选项 |
| **D. 与NPC交流** | 深入了解三人，加深羁绊 | 影响去留 |
| **E. 调查圣焰** | 为第二幕铺垫（焕的线索）| 自由探索 |
| **F. 其他** | 玩家自由行动 | AI响应 |

> *（你也可以不说选项里的内容——用自己的方式描述你想做的事。）*

---

### 阶段二：下午（可选——玩家第二个决策或跳过）

晨间决策执行完毕后，过渡到下午：
```
"时间过得很快。太阳开始西斜，酒馆还没到开门时间。
下午的光线从窗户透进来，雨在检查食材，焕不知道什么时候出去了还没回来。"
```

**下午选项**：
- 如果晨间做了某件事，想继续深入 → 推进该线索
- 如果想处理上午没处理的事 → 同样可以
- 如果想跳过 → 直接进入傍晚

```
> 下午你想做什么？
> 
> [1] 继续上午的事（深入）
> [2] 做点别的
> [3] 跳过，直接到傍晚准备营业
> 
> *（你也可以不说选项里的内容——用自己的方式描述。）*
```

**跳过规则**：如果玩家选择 [3] 或描述"直接营业"，DM 直接推进到晚间结算，不强制要求两个决策。

---

### 阶段三：傍晚—晚间结算

酒馆开门营业，DM 简短描述当晚情况（1-2句），然后给出结算结果。

---

### 每日决策与检定情境

每天的决策中，玩家可能主动触发 d20 检定。DM 根据玩家描述的行动类型决定是否需要检定。

**可触发检定的情境（玩家主动尝试时）**：

| 情境 | 检定类型 | DC | 说明 |
|------|----------|-----|------|
| 和供应商/鱼贩讨价还价 | 魅力(交易) | 12 | 成功可降低成本或获得更好货源 |
| 说服/威压闹事者 | 魅力(威压) 或 魅力(说服) | 13-15 | 视对方强硬程度 |
| 在竞争对手面前立威 | 魅力(表演) | 14 | 成功打响名声 |
| 处理挑剔客人的投诉 | 心智(观察) 或 魅力(说服) | 12 | 成功挽回口碑 |
| 尝试新菜谱/创新菜 | 心智(烹饪) | 13 | 成功提升菜品质量加成 |
| 突发事件：有人斗殴 | 体魄(格斗) 或 敏捷(隐匿) | 14 | 成功平息；失败可能引发更大麻烦 |
| 偷偷调查某人/某事 | 敏捷(隐匿) 或 心智(观察) | 12-15 | 视调查目标的警觉程度 |
| 抢在竞争对手前争取资源 | 敏捷 vs 对手敏捷 | 对手而定 | 快进式竞争 |

**明骰格式（必须展示）**：
```
d20([掷骰数字]) + 属性修正(+N) = [合计] vs DC [目标]
→ 成功 / 失败 / 大成功 / 大失败
```

**触发原则**：
- 玩家说"我要尝试 X"时，判断是否有风险/阻碍
- 纯叙事、没有对抗的选择 → 不需要检定，直接叙事结果
- 有 NPC 主动对抗/阻拦时 → 触发检定
- "我想试试"时 → 邀请玩家描述具体怎么做的，再判断

---

### 晚间结算

每天结束时，AI DM 根据以下因素给出一个**结果**：

**计算公式（简化版）**：

```
当日收入 = 基础客流 × 菜品质量加成 × 事件修正
```

| 因素 | 影响 |
|------|------|
| **菜品质量** | 取决于食材新鲜度。雨的能力是固定的（除非升级）。 |
| **基础客流** | 固定 50-70 铜/天（伊格尼斯下层街区，流量有限）|
| **食材新鲜度** | 好食材 +30% 收入；普通食材 ±0%；差食材 -20% |
| **事件修正** | 有利事件 +20-50%；不利事件 -10-30% |

**结果分档**：

| 结果 | 描述 | 效果 |
|------|------|------|
| **大成功** | 客满，口碑传播，有人多给小费 | +2天连续达标进度 |
| **达标** | 稳定收入，能维持 | +1天连续达标进度 |
| **勉强** | 收支平衡，但明天需要改变 | +0天连续达标进度 |
| **亏损** | 入不敷出，有人抱怨 | -1天连续达标进度（倒退）|

## 🍽️ 食材系统（简化）

### 食材来源

| 来源 | 食材质量 | 价格 | 风险 |
|------|----------|------|------|
| **正规市场** | 普通 | 正常价 | 无 |
| **走私者通道** | 好（最新鲜）| 便宜 30% | 可能遇到麻烦 |
| **港口早市** | 好（有时有好货）| 正常价 | 无 |
| **老猪窝施舍** | 差 | 不要钱 | 丢面子，影响口碑 |

### 食材储备

酒馆初始有**3天份的基础食材储备**（前老板留下的，勉强能用）。

雨每天消耗1份食材。食材耗尽时必须补充，否则菜品质量下降。

---

## 👥 NPC贡献（每日自动生效）

| NPC | 贡献 | 负面影响 |
|-----|------|----------|
| **雨** | 稳定出品，菜品质量有保障 | 无——除非玩家让她不高兴 |
| **焕** | 守护酒馆安全，应对骚扰 | 无——除非触发他的过去 |
| **利希特** | 吉祥物，吸引客人（真的有用）| 无——只要有鱼吃 |

**NPC心情系统**（隐性）：

三人都有隐藏的"满意度"状态。玩家的行为会影响它：

| 行为 | 影响 |
|------|------|
| 认真做事、扛责任 | 三人满意度+ |
| 回避问题、犹豫不决 | 三人满意度- |
| 照顾到某人的需求 | 那个人的满意度大幅+ |
| 伤害/忽视某人 | 那个人的满意度大幅- |

满意度长期低迷可能导致NPC离开。这是玩家决策的后果之一。

## ⚠️ 每日事件系统

每天由 DM 主观判断是否触发事件（0-1个），不要机械查表。

**DM 判断标准**：
- 剧情需要时（玩家做了值得奖励/惩罚的事）→ 加事件
- 感觉今天太平淡 → 加一个中性/正面事件调剂节奏
- 玩家已经连续两天无事发生 → 第三天加个事件

### 正面事件

| 事件 | 效果 |
|------|------|
| **口碑传开** | 当日客流+20%，大成功更容易 |
| **有人介绍新供应商** | 解锁走私者通道采购（质量好，便宜）|
| **老猪窝失误** | 他们的客人流向了灰烬酒馆 |
| **焕赶走了骚扰者** | +焕的满意度，触发他的能力展示 |

### 负面事件

| 事件 | 效果 |
|------|------|
| **老猪窝造谣** | 当日客流-30% |
| **供应商拒绝供货** | 必须去别的地方采购，耽误时间 |
| **美食协会暗访** | 提前触发！当天结算按暗访标准 |
| **有人喝醉闹事** | 需要处理，否则影响第二天 |

### 中性事件

| 事件 | 效果 |
|------|------|
| **神秘客人** | 付钱很多，但留下奇怪的话（第二幕伏笔）|
| **有人问起"圣焰"** | 与焕有关的线索出现 |
| **利希特叼回来一条稀有鱼** | 食材质量大幅提升 |

## 🏆 达标里程碑

### 连续3天达标

当玩家达成 **连续3天达标**（任意组合：大成功/达标都可以）时：

```
→ 触发"资格确认场景"（act1_qualification_zh.md）
→ 美食大赏资格到手
→ 第一幕目标达成
```

### 中间断开

如果中间有**亏损**：
- 连续计数归零
- 但不会立刻失败——给玩家1-2天缓冲期来调整

如果**连续2天亏损**：
- 触发"危机场景"——三人坐下来谈，玩家需要证明自己值得跟
- 证明不了，雨会说"我就知道你不行"，焕和利希特可能也会离开

## 📋 AI DM 每日结算模板

### 达标日（示例）

```
傍晚，最后一位客人离开了。

今天的经营结束了。雨在厨房里洗碗，焕在角落里靠着墙闭目养神，利希特在窗台上已经睡着了。

"今天……还行。"雨的声音从厨房传来，带着一丝不情愿的满意，"食材一般，但味道没丢。——别得意，明天要是搞砸了，我一样骂你。"

你们一共接待了XX位客人，收入XX铜。连续达标天数：1/3天。

明天又是新的一天。
```

### 亏损日（示例）

```
傍晚，雨站在吧台后面，表情很冷。

"今天的客人……"她擦着杯子的动作慢了下来，"比昨天少了一半。老猪窝那边又在造谣，说我们用过期的东西。"

焕从角落里睁开眼睛，扫了你一眼，又闭上了。

雨放下杯子，声音低了一点：

"明天得想办法了。不然……"她顿了顿，"不然我存的那点私房钱，可能要先垫上了。"

连续达标天数：0/3天。明日危机等级：⚠️
```
