# 第一幕 · 资格确认场景

> **场景类型**：第一幕高潮 / 收尾
> **触发条件**：连续3天达成日收入目标
> **预计时长**：5-10分钟
> **目的**：确认资格、团队羁绊建立、第二幕过渡

---

## 触发时机

当玩家的连续达标天数达到 **3/3天** 时，在当天晚间结算后触发本场景。

---

## 📜 叙事文本

### 傍晚 — 最后一桌客人

```
最后一桌客人离开了。

雨从厨房里走出来，手里端着一盘她自己也没吃过的小菜——
是她刚研发的新菜式，用来庆祝今天的。

"……试试。"她把盘子放在你面前，没看你。

焕从角落里站起来，走到桌边，没说话，但站得比平时近了一点。

利希特从窗台跳下来，叼着它今天存下的一小块鱼干，放在盘子旁边，
然后抬头看你，一只眼睛终于完全睁开了。

你们三个——不，四个人——围坐在这张破旧的桌子旁边。
灰烬酒馆从来没有这么……像样过。
```

---

### 夜晚 — 敲门声

```
有人敲门。

雨去开门。门口站着一个穿着体面长袍的中年人，
手里拿着一张盖有火红色印章的纸。

"灰烬酒馆？"他看了看里面的环境，眉毛微微挑了一下，
但很快恢复了职业性的微笑。

"美食协会巡访员。你们这三天的营业记录符合参赛标准。"

他把纸放在桌上。

印章是火红色的，上面刻着一朵跳动的火焰纹样。

"圣焰美食大赏，预选赛资格——确认。"

他转身就走，没有多说一句话。

门关上。雨盯着那张纸，沉默了很久。
```

---

### 深夜 — 雨的坦白

```
那天晚上，酒馆打烊后。

焕已经在角落里躺下了，利希特在窗台上睡着了。
只有雨还站在吧台后面，擦着那只永远擦不干净的杯子。

你正准备上楼休息，她突然开口了。

"……谢谢你没有跑。"

她的声音很轻，像是说给自己听的。

"前老板走的时候，他说第二天就回来。
结果第二天人就不见了。
打电话，没人接。去他说的地址，是个空房子。"

她放下杯子，终于看向你。

"我等了一个月。"

她没有再说下去。但你懂了。

她不是在等一个老板。
她是在等一个不会跑的人。

你转身的时候，她的声音又响起来：

"……明天的菜，我来想办法。协会的人可能会提前来。"

她的语气已经恢复了平时的刺。

但那一瞬间的松动，你知道是真的。
```

## 💡 场景说明

### 这一刻的重要性

这是第一幕的情感高潮。三天的经营不只是"游戏进度"，而是**四个人从陌生人变成团队**的过程。

雨的那段坦白——"我等了一个月"——是她全剧最脆弱的一刻。玩家在这里知道了她为什么那么怕被抛弃。

### 焕的沉默

焕没有说话。但他站得比平时近了，把利希特的鱼干放在盘子旁边——这是他表达认同的方式。不需要台词。

### 利希特

海豹的"正式睁眼"是一种郑重。平日里它总是半睡半醒，但这一刻它认真了。
