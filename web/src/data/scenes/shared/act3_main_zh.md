## 场景概览

**时间**：献祭仪式前夕
**地点**：灰烬酒馆 / 圣焰神殿
**核心事件**：玩家做出最终选择，故事收尾

---

## ⚖️ 第三幕核心：电车难题

**背景**：玩家已决定是否阻止献祭。
- **如果选择阻止** → 献祭将不会发生，但城市将失去圣焰的保护（后果未知）
- **如果选择接受** → 又一条生命将被献祭，换取城市继续繁荣

**最终选择没有"正确答案"**。

### 三种结局路线

#### 结局 A：救家人（阻止献祭）

**触发条件**：玩家选择阻止献祭

```
焕的眼中闪过一丝光。

「……好。」

他抬起左手——手背上的蓝绿色纹路突然亮起。

「那就让这场火……烧在我身上。」

焕用自己的火焰（契约之火）点燃了圣焰神殿。
深渊裂缝被短暂打开——但利希特站了出来。

「……使命完成了。」

海豹的身体开始发光。
它跃入裂缝，用自己的神圣力量封印了深渊。

圣焰继续燃烧——但这一次，不是以生命为代价。

代价是：利希特失去了神力，变成了普通的海豹。
但它活了下来。
```

#### 结局 B：救城市（接受献祭）

**触发条件**：玩家选择接受献祭（或无作为）

```
雨从厨房里走出来，手里端着一碗汤。

「……吃吧。」

她的声音很轻。「不管怎样……先吃饱。」

焕沉默地看着火焰。

「……我会去。」

他说。「这一次，换我自己选。」

他走进了圣焰神殿。

光芒吞噬了他。

圣焰继续燃烧。又一个十年。
```

#### 结局 C：找到第三条路（特殊条件解锁）

**触发条件**：玩家在 Act II 中收集了足够的线索 + 做出特定选择

```
利希特从窗台上跳下来，走到桌子中央。

它抬起头，一只眼睛完全睁开了——金色，圣洁。

「……至高女神，请赐予我最后的力量。」

它的身体开始发光。

「等等——」焕的声音突然变了，「它不是普通海豹——它是……」

利希特跃入圣焰。

光芒没有吞噬任何人。

圣焰熄灭了——但这一次，是真正的熄灭，不是契约的延续。

深渊裂缝永远关闭了。

代价是：伊格尼斯失去了"圣焰认证"的光环，城市开始衰退。
但这里的人……活了下来。

你们离开了伊格尼斯。
灰烬酒馆关了门。
但你们四个——还在。
```

## 🎭 各角色结局

| 角色 | 结局 A（救家人）| 结局 B（救城市）| 结局 C（第三条路）|
|------|----------------|----------------|----------------|
| **雨** | 终于不再害怕失去——她有了真正的家 | 留下，继续经营酒馆（没有你们）| 带着酒馆的食谱离开，和大家一起走 |
| **焕** | 放下了复仇，与过去的火焰和解 | 选择了牺牲，完成了自己的使命 | 失去了契约之力，但找回了自己 |
| **利希特** | 失去神力，变成普通海豹，但活了下来 | 见证了牺牲，继续自己的使命 | 用神力封印了深渊，完成了族人的愿望 |

## 🔚 尾声模板

**结局 A 尾声**：

```
一年后。

伊格尼斯的圣焰熄灭了。
城市开始衰退，但人们还在。

你们四个离开了那座城市。
灰烬酒馆——在另一个地方重新开了门。

「欢迎光临。」

雨站在厨房里，焕靠在角落的墙边。
利希特在窗台上晒太阳——它再也不会发光了。

但它的眼睛，还是只睁着一只。

「……今天的特色菜是辣味鱼。」雨说。

焕微微点头。

利希特发出满足的鼻息声，鱼袋还挂在腰间。

你们还在。
这就是结局。
```
