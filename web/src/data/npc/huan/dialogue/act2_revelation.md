## 事件 2：焕讲述十年前的真相

**焕的讲述**（分3段，AI DM 分次揭示）：

### 第一段：我是谁

```
「十年前，我住在一个叫灰泉镇的地方。」

焕的声音很平静。

「我是那个地方——最后的幸存者之一。」
```

### 第二段：发生了什么

```
「献祭不是自愿的。」

焕的金瞳直视着某处遥远的地方。

「城市需要繁荣。需要火焰。需要契约。」

「我的家乡——137个人——就是那个契约的代价。」

他的手背上，蓝绿色的纹路微微发光。

「我本来也应该死在那场火里。但我没有。父母把我推进了地窖——我和妹妹。」

沉默。

「……后来我们走散了。我再也没有找到她。」
```

### 第三段：我为什么在这里

```
「我在找答案。也在找她。」

焕沉默了一会儿。

「两年前，我在一场战斗中濒死。有东西向我伸出手——我抓住了。代价是一部分灵魂。」

他看着自己的左手。

「这火焰……它能感应到某种东西。把我引到了这里。我不确定它指向什么，但我无法忽视。」

「守焰者——是执行契约的人。他们在城市里。协会的背后……就有他们的人。」

「……我留下来，是因为这个地方——藏着我要找的东西。」
```
